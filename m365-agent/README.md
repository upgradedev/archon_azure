# Archon — Microsoft 365 Copilot Declarative Agent

> **Microsoft Agents League — Enterprise Agents Track**
> Microsoft IQ: **Foundry IQ** (Azure AI Search grounding in narrator agent)

This directory contains the Microsoft 365 Copilot declarative agent that brings Archon's financial intelligence into Teams, Copilot Chat, and Microsoft 365.

---

## What it does

An employee or business owner opens **Teams** or **Microsoft 365 Copilot Chat** and asks:

> *"What was our P&L for January 2026?"*
> *"What is our true payroll cost including IKA contributions?"*
> *"Give me the executive summary for last month."*

Archon calls the backend API, runs the 6-agent analysis pipeline (if needed), and returns a grounded financial answer — with IFRS / Greek regulation citations powered by **Foundry IQ**.

## Deployment steps

### Prerequisites

- Microsoft 365 tenant with Copilot licence (Microsoft 365 Copilot or Teams Premium)
- Admin access to **Microsoft Teams Admin Center** or **Microsoft 365 Admin Center**
- Archon backend deployed and publicly reachable (see `infra/main.bicep`)
- App registration in **Microsoft Entra ID** for auth

### Step 1 — Register an App in Entra ID

```bash
# Create app registration (returns appId)
az ad app create \
  --display-name "Archon Financial Intelligence" \
  --sign-in-audience AzureADMyOrg

# Note the appId, then expose an API scope:
# API permissions → Expose an API → Add scope: archon.read
```

### Step 2 — Update the API server URL

Edit `openapi.json` → `servers[0].url` → replace with your deployed backend URL:

```json
"servers": [
  {
    "url": "https://archon-api.YOUR_ENV.azurecontainerapps.io"
  }
]
```

### Step 3 — Package the agent

Zip the three files together (flat, no subdirectory):

```bash
cd repos/azure/m365-agent
zip archon-m365-agent.zip manifest.json openapi.json assets/icon-color.png assets/icon-outline.png
```

### Step 4 — Sideload via Teams Admin Center

1. Go to [Teams Admin Center](https://admin.teams.microsoft.com) → **Teams apps** → **Manage apps**
2. Click **Upload** → choose `archon-m365-agent.zip`
3. Once uploaded: **Apps** → **Archon Financial Intelligence** → **Publish**
4. Assign to users or groups via **Permission policies**

### Step 5 — Test in Copilot Chat

Open Microsoft 365 Copilot Chat → click the **+** (plugins) → enable **Archon Financial Intelligence** → try the conversation starters.

## Authentication

The agent uses **Microsoft Entra ID** (bearer token) to authenticate API calls to the Archon backend. The backend validates the token via the `Authorization: Bearer` header.

For demo purposes, you can temporarily relax auth on the backend (`/api/analyze` open endpoint) and remove the `security` block from `openapi.json`.

---

## Foundry IQ integration

The executive summary returned by `analyzeFinancials` is generated by the **NarratorAgent** (see `endpoints/analysis/agents/narrator.py`). In production it uses the **azure-ai-projects** SDK with **AzureAISearchTool** to retrieve IFRS standards and Greek payroll regulations before writing the summary.

This grounds the M365 Copilot response in authoritative, cited knowledge — not LLM hallucination.

## Architecture

```
Microsoft 365 Copilot Chat / Teams
        │ declarative agent (manifest.json)
        │ OpenAPI plugin (openapi.json)
        ▼
Archon FastAPI Backend (Azure Container Apps)
  /api/analyze  →  6-agent pipeline
  /api/reports  →  cached reports
        │
        ▼
Azure AI Foundry (NarratorAgent)
  AzureAISearchTool → archon-knowledge index
  (IFRS · IKA/EFKA · VAT regulations)
        │
        ▼
Azure OpenAI GPT-4o
  Regulation-cited executive summary
```
